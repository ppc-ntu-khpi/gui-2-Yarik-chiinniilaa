### 1. Мета роботи
Навчитися швидкої розробки складних графічних інтерфейсів за допомогою візуального редактора Matisse у середовищі NetBeans. Ознайомитися з принципами автоматичної генерації Swing-коду, роботою менеджерів автоматичного розташування компонентів, конфігуруванням властивостей елементів (таких як `resizable`) та зв'язуванням дій користувача з бізнес-логікою.

---

### 2. Критерії виконання лабораторної роботи

* **На "трійку":** Створено візуальний прототип форми за допомогою конструктора Matisse. Налаштовано властивість форми `Resizable = false` для блокування некоректного розтягування вікна. Підключено зовнішню бібліотеку класів `MyBank.jar`.
* **На "чотири":** Інтегровано модуль `DataSource` для автоматичного парсингу файлу `data/test.dat`. Зв'язано випадаючий список із базою клієнтів. Створено обробник події кліку на кнопку **Show**, який виводить інформацію про **усі рахунки** обраного користувача. Додано кнопку **About**, що викликає інформаційне модальне вікно `JOptionPane`.
* **На "п'ять":** Додано інтерактивну кнопку **Report**, яка генерує зведений фінансовий звіт за формою `CustomerReport` з лабораторної роботи №8 та відображає його у моноширинному форматі в центральній текстовій зоні.
